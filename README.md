# 🧠 Plataforma de Detección Temprana Emocional

Una aplicación Streamlit para evaluar y monitorear el bienestar emocional de estudiantes.

## ✨ Características

- ✅ **Evaluación adaptativa** por nivel educativo (Primaria, Secundaria, Universidad)
- ✅ **Análisis emocional** con TextBlob y POMS (Profile of Mood States)
- ✅ **Clasificación de riesgo** automática (Alto, Medio, Bajo)
- ✅ **Dashboard interactivo** con Plotly
- ✅ **Panel docente** con clustering de riesgo (K-Means)
- ✅ **Alertas inteligentes** basadas en tendencias
- ✅ **Exportación de datos** (CSV, Excel, PDF, ZIP)
- ✅ **Base de datos SQLite** local y segura
